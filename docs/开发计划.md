# FactorAI 开发计划（V1）

## 里程碑（当前状态）

### M1：文档与框架搭建（已完成）

- PRD、技术架构、节点协议、开发计划
- 前端工程初始化

### M2：核心页面开发（已完成）

- 首页

- 模板库
- 编辑器（节点库/画布/属性）

- Python 节点沙箱执行（超时、内存限制、字段校验）
- 前端接入真实 API 并完成轮询刷新

### M5：质量验证（进行中）

- TypeScript 构建
- 代码规范检查

### M6：后续演进（下一阶段）

- 后端 API 接入
