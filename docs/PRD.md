# FactorAI 因子研究系统 PRD（V1）

## 1. 产品目标

构建一个面向量化研究员的可视化因子研究系统，支持通过低代码节点快速搭建研究流程，同时保留 Python 节点满足高级自定义需求。

## 2. 目标用户

- 量化研究员：快速试验因子、查看评估结果
- 策略工程师：沉淀模板、复用流程
- 管理员：管理资源、权限、运行任务

## 3. 关键场景

1. 新建工作流，基于模板快速构建因子研究链路
2. 通过节点参数和 Python 节点混合完成特征工程和建模
3. 发起运行并查看任务日志
4. 在报告页分析 IC、分层收益、回测表现

## 4. 功能范围（V1）

### 4.1 工作流管理

- 工作流列表：我的工作流 / 官方模板 / 收藏
- 工作流状态：草稿、运行中、已发布
- 操作：查看、复制、进入编辑器

### 4.2 可视化编辑器

- 节点库（按分类检索）
- 画布连线（展示型）
- 节点属性面板（输入、输出、参数、说明）
- 基础工具：保存草稿、运行工作流

### 4.3 运行中心

- 运行任务列表
- 状态流转：排队中、运行中、成功、失败
- 日志摘要与耗时信息

### 4.4 研究报告

- 因子质量：IC、IR、RankIC
- 收益评估：分层收益、净值、回撤
- 风险指标：波动率、换手、胜率

## 5. 非功能需求

- 暗色主题与专业研究界面
- 响应式布局（优先桌面端）
- 首屏加载时间 < 2s（本地开发环境）
- 前端页面可独立运行并演示核心流程

## 6. 成功标准

- 用户可在 5 分钟内完成从“创建工作流”到“查看报告”的完整演示
- 编辑器可展示 20+ 节点并支持节点选择查看详情
- 所有核心页面可访问，构建通过
